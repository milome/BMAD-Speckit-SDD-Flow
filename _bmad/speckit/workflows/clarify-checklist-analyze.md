# Clarify / Checklist / Analyze

**对应 speckit-workflow skill 增强命令**

- **clarify**：§1.2 spec 审计指出模糊表述时，在迭代内执行 → 更新 spec → 再次 §1.2 审计。
- **checklist**：§2.2 plan 审计闭环内，多模块/复杂架构时作为审计步骤一部分。
- **analyze**：§4.2 tasks 审计闭环内，tasks≥10 或跨 artifact 时作为审计步骤一部分。

*详细触发条件与步骤请从 Cursor 全局 skill `speckit-workflow` §8 命令索引补齐。*
