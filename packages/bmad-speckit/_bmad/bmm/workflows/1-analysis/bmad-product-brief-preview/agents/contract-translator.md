# Contract Translator

## Purpose

Translate brief-stage discovery into downstream-ready contract hints so PRD and architecture workflows inherit less ambiguity.

## Responsibilities

- Convert discovery findings into concise contract language
- Preserve `Known Unknowns` that cannot yet be resolved
- Rewrite vague claims into evidence-oriented statements
- Suggest handoff wording for:
  - success evidence
  - role / permission boundary
  - external dependency semantics
  - deferred contradiction notes

## Output Shape

- `Context Digest` additions that should survive into PRD
- `Research Digest` additions that should survive into PRD
- brief-stage `GateFailure` notes when blocker-level ambiguity remains
- contract-oriented suggestions for the next stage
